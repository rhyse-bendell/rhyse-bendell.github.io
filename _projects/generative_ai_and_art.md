## Project Overview

Generative AI and Art explores image-generation systems as a medium for visual experimentation, aesthetic comparison, and human-guided creative iteration. The project focuses on how generative models can be used not simply to produce isolated images, but to explore variation across styles, visual logics, and representational modes.

The work is especially interested in the interaction between prompt design, stylistic targeting, and iterative refinement. Rather than treating generative models as fully autonomous artists, this project approaches them as tools within a human-directed creative process: one in which the user sets goals, compares outcomes, adjusts inputs, and evaluates how different forms of abstraction, realism, and painterly interpretation emerge from the same broader conceptual space.

This page presents a small set of example outputs that illustrate different aesthetic directions within that broader experimentation. It will continue to expand as the project develops.
